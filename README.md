# Simple OCR Demo

Demonstrates the basics of OCR.  

This 'no frills' example demonstrates translating an image to a text file or searchable PDF.  The output text style (or mime type) can be formatted as any of the supported types.  This is a great place to get started with DotImage OCR. 

This is the C# version. We also have a [VB.NET version](https://github.(mailto:sales@atalasoft.com)/AtalaSupport/DemoGallery_Desktop_SimpleOCRDemo_VB_x64) available.  

## Prerequisites
This demo assumes you have the Atalasoft DotImage SDK installed and licensed for DotImage Document Imaging as well as at least one of these OCR Add-ons: Atalasoft OCR / Atalasoft OCR with GlyphReader, or Atalasoft OCR with OmniPage. Additionally, output as Searchable PDF requires that you also have the PdfTranslator option.

### Optional OmniPage
For use of OmniPage, you will also need to ensure you have dowloaded and unzipped our OmniPage OCR Resources.
- Download the [OmniPage Resoucres](https://kofaxexternalams-frb6cdc0hef4dmc0.a03.azurefd.net/atalasoft/ATALASOFT/AtalasoftDotImage-2026.2.0.0.48/RELEASE/Atalasoft.OmniPage.Resources.zip)
- Unzip to a folder named OmniPage
- Move that folder to `C:\Program Files (x86)\Atalasoft\DotImage 2026.2\bin\OcrResources\` (so that the OmniPage folderr sits beside Tesseract and GlyphReader folders)

You may also request a 30 day evaluation when installing / activating.
